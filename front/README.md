# MCP Gateway Frontend

Modern dashboard for the MCP Gateway, built with Next.js 15, Tailwind CSS, and TypeScript.

## Features

- **Dashboard**: Overview of your MCP infrastructure status.
- **Server Management**: Detailed inspection of tools, resources, and prompts.
- **Health Monitoring**: Real-time health metrics.
- **Theme**: Vercel-inspired dark mode aesthetic.

## Tech Stack

- Next.js 15 (App Router)
- Tailwind CSS
- Lucide React (Icons)
- TypeScript

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Open [http://localhost:3000](http://localhost:3000)

## Project Structure

- `src/app`: App Router pages
- `src/components`: UI components
- `src/lib`: Utilities and mock data
